# 规划功能（待实现）

## 1. 模型实时状态指示器

每个模型条目显示一个状态圆点：
- 🟢 **绿灯闪烁** — 该模型当前正在被使用（有请求正在通过该模型处理）
- ⚫ **灰点** — 该模型空闲，未被使用

### 需要实现
- **后端**：跟踪每个模型当前的活跃请求数，暴露状态接口
- **前端**：轮询或 SSE 实时更新圆点状态，绿灯用 CSS animation 闪烁
- 相关文件：`server.rs`、`ModelPool.tsx`

## 2. Token 用量统计

- 跟踪**每个模型**消耗的 token 数（输入 + 输出）
- 跟踪**总计** token 消耗
- 在 UI 中展示（每个模型行旁边 + 总览区域）

### 需要实现
- **后端**：在代理转发请求时解析 usage 字段，累计到模型维度的计数器
- **前端**：展示每个模型的 token 用量和总量
- 相关文件：`server.rs`、`ModelPool.tsx`
